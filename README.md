# 项目简介

一款Vue+Less+Pinia+Vue Router+Axios+Element Plus通用脚手架，具备基本的登录注册页面,可以快速启动并搭建通用Vue项目

## 项目启动
### 1、安装依赖

```sh
npm install
```

### 2、项目启动

```sh
npm run dev
```

### 3、项目打包

```sh
npm run build
```
